# 🚀 Complaint Desk System

A full-stack MERN application to manage and track user complaints efficiently.

---

## 🔥 Features
- 👤 User Registration & Login
- 📝 Submit Complaints
- 📊 Admin Dashboard
- 🔍 Track Complaint Status

---

## 🛠️ Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
